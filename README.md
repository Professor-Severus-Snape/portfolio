# Моё портфолио

1. Данная работа представляет собой SPA (React + TypeScript).
1. Для проекта мной впервые самостоятельно разработано 3 полноценные версии макета в figma.
1. Впервые реализована `mobile-first` вёрстка.
1. В проекте использован препроцессор `SASS` в сочетании с БЭМ-методологией.
1. Освоена работа с библиотекой `classnames` для создания динамических классов.
1. Сделан упор на мемоизацию данных в компонентах.
1. Упрощен импорт за счёт использования индексных файлов и алиасов.
1. Валидация кода - через [валидатор HTML](https://validator.w3.org/) и [валидатор CSS](https://jigsaw.w3.org/css-validator/).
1. Проверка производительности - через [PageSpeed Insights](https://pagespeed.web.dev/).
1. Авторские права защищены лицензией MIT. Любое использование данного проекта должно сопровождаться уведомлениями об авторских правах и лицензией.
